# HR Analytics Dashboard (Power BI)

## Overview
This project is a professional HR Analytics Dashboard built using Power BI. It analyzes employee data to provide insights into workforce distribution, salary trends, and employee attrition.

## Features
- KPI Metrics: Total Employees, Attrition Rate, Avg Salary, Active Employees
- Interactive Dashboard with slicers (Department, Gender, Job Role)
- Visual Analysis:
  - Employees by Department
  - Employees by Job Role
  - Gender Distribution
  - Education Field Analysis

## Tools & Technologies
- Power BI
- DAX
- Data Cleaning (Power Query)

## Dataset
IBM HR Analytics Dataset (Kaggle)

## Insights
- Identified departments with higher employee concentration
- Highlighted attrition patterns across roles
